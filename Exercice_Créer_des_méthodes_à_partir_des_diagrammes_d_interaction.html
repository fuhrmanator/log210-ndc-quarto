Pour chacun des diagrammes suivants, écrire les classes TypeScript avec les méthodes indiquées dans le diagramme. La matière du livre est à la section F18.6/A20.4.

Avertissement

Vous pouvez utiliser VSCode pour vous aider avec le TypeScript, mais ça ne sera pas possible à l’examen final

Modèle :

class A {
    b: B;  // A envoie un message à B, visibilité d'attribut
    execute(arg0:number):any {
        const result = this.b.setItem("Fred");
    }
}

class B {
    setItem(arg0:string):any {
        //...
    }
}
